## Tipos de números (#1)

Adivina los resultados de las siguientes expresiones y luego compruébalos usando Kotlin:

```kotlin
val a = 11 / 3
val b = 11 % 3
val c = a * 3 + b
val d = 6 / 5.0
```

Ten en cuenta que los `val`s `a`, `b` y `c` son de tipo `Int`, mientras que el `val` `d` es de tipo `Double`.
