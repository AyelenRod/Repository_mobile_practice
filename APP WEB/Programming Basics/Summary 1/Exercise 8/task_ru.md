## Резюме 1 (#8)

Переверните цифры в десятичном числе, используя цикл `while`. Вспомогательная переменная `var worker` поможет вам итеративно создать `result`:

| Worker     | Result |
|------------|--------|
| 1234       | 0      |
| 123        | 4      |
| 12         | 43     |
| 1          | 432    |
| 0          | 4321   |
