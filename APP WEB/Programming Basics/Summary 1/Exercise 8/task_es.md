## Resumen 1 (#8)

Invierte los dígitos de un número decimal usando un bucle `while`. Una `var worker` auxiliar te ayudará a producir iterativamente el `resultado`:

| Trabajador | Resultado |
|------------|-----------|
| 1234       | 0         |
| 123        | 4         |
| 12         | 43        |
| 1          | 432       |
| 0          | 4321      |
