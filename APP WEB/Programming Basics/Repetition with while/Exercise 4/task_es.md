## Dominando el IDE: mover el cursor

Puede mover el cursor una palabra a la vez. Lo mismo funciona para seleccionar texto:

| Operación                              | Atajo                                                                   | Descripción                                       |
|----------------------------------------|------------------------------------------------------------------------|---------------------------------------------------|
| Mover a la siguiente palabra           | <span class="shortcut">&shortcut:EditorNextWord;</span>                | Mover el cursor a la siguiente palabra.           |
| Mover a la palabra anterior            | <span class="shortcut">&shortcut:EditorPreviousWord;</span>            | Mover el cursor a la palabra anterior.            |
| Mover a la siguiente palabra y seleccionar | <span class="shortcut">&shortcut:EditorNextWordWithSelection;</span> | Mover el cursor a la siguiente palabra, seleccionándola. |
| Mover a la palabra anterior y seleccionar | <span class="shortcut">&shortcut:EditorPreviousWordWithSelection;</span> | Mover el cursor a la palabra anterior, seleccionándola. |

Seleccione `getFooResult()` moviendo el cursor sobre las palabras y extráigalo en un
`val foo` manualmente. A continuación, discutiremos cómo extraer una variable automáticamente.
